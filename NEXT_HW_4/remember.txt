pwd로 쉽게 경로를 확인할 수 있다.
cd ..을 통해 이전 파일 경로로 접근할 수 있다.
mkdir 폴더 이름을 통해 쉽게 폴더를 만들 수 있다.
rm 파일 이름을 통해 파일을 쉽게 지울 수 있다.
원격저장소를 remote repo라고 한다. 내 컴터에 있는 건 local repo이다.
git clone을 통해 코드 복제 및 연결. git remote add origin ${repo 주소}를 통해 local과 remote를 연결.
git add한 것을 취소하고 싶으면, git reset ${파일 이름}을 통해 취소할 수 있다.
git merge ${머지하고 싶은 브랜치}
